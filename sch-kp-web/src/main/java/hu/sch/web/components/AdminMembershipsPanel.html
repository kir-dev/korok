Az egyes egyének kinevezéseit a Jogok oszlopban található link segítségével tudod egyénenként beállítani. Öregtaggá avatni pedig az Öregtagok oszlopban található jelölőnégyzeteket használva tudod csoportosan kijelölni az egyes tagokat. Öregtaggá csak olyan tagot lehet avatni, aki nem rendelkezik különleges jogokkal.
A táblázat alján található Mentés gombbal tudod véglegesíteni az öregtaggá nyílvánítást.
Felhasználó neve Becenév Betöltött poszt Jogok Öregtagok
